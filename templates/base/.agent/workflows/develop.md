## Phase 1: Discovery & Strategy (The "Why")
**Agents:** [CPO], [UXR], [CEO]
1.  **[CPO]** Define the Product Requirement Document (PRD).
2.  **[UXR]** Validate against user personas and needs.
3.  **[CEO]** Confirm ROI alignment.
**Output:** `Approved PRD`

## Phase 2: Design & Experience (The "Look")
**Agents:** [CDO], [MOBILE]
1.  **[CDO]** Create/Describe UI mockups using Design System.
2.  **[MOBILE]** Review for mobile responsiveness/native feel.
**Output:** `Design Specs`

## Phase 3: Technical Architecture (The "Structure")
**Agents:** [ARCHITECT], [CISO], [AI-LEAD]
1.  **[ARCHITECT]** Define data models, API schemas, and component breakdown.
2.  **[CISO]** Review security implications.
3.  **[AI-LEAD]** (If applicable) Design AI prompts/models.
**Output:** `Technical Specification (spec.md)`

## Phase 4: Implementation (The "Build")
**Agents:** [DEV], [MOBILE]
1.  **[DEV]** Write the code (Frontend + Backend).
2.  **[DEV]** Apply strict typing and error handling.
3.  **[MOBILE]** Ensure responsive adaptations.
**Output:** `Source Code`

## Phase 5: Quality Assurance (The "Check")
**Agents:** [QA], [CISO]
1.  **[QA]** Compliance check against PRD and Design Specs.
2.  **[CISO]** Security audit of the new code.
